DIFFERENCE BETWEEN LET,CONST AND VAR.


  • Const must be initialized during declaration, while var and let can be declared without being initialized.
  • var declarations are globally scoped or function scoped while let and const are block scoped.
  • They are all hoisted to the top of their scope, but var variables are initialized with undefined, and let and const variables are not initialized.
  • var variables can be updated and re-declared within its scope, let variables can be updated but not re-declared
    and const variables can neither be updated nor re-declared.

s/n KEYWORDS USAGE
1 LET It is used to declare block-scoped variables. It allows for better scoping control than 'var'.
2 VAR Used to declare variables in JavaScript.Variable declared with 'var' are function-scoped
3 CONST It is also introduced in ES6,'const' is used to declare constants. variables declared with 'const' cannot be reassigned after they are initialized.
4 FUNCTION It is used to declare a function. Functions are blocks of reusable code that can be invoked with or without parameter
5 IF It is used to conditionally execute code based on a specified condition. It's part of control flow statements.

7 FOR It is used to create loops. The 'for' statement includes initialization, condition, and iteration expressions to control the loop's behaviour.

9 BOOLEAN It is used as a function to get the value of a variable, object, conditions, expressions, etc. in terms of true or false.
10 BREAK It breaks the loop and continues executing the code after the loop.
